Instructions to compile and run divide and conquer tsp

Ant is used to both run and compile the application.

1. Compiling:
Run the command ant to compile program
The application has five packages:
- api
- clients
- system
- tasks
- util

2. Running the code:

Space/server:
2.1 Start the space server by running the command ant runSpace -Darg0="[true/false: true if space runnable tasks activated, false otherwise]"

Computer(s):
2.2 Start computer server(s) by running the command ant runComputer -Darg0="[ip of server machine]" -Darg1="[true/false: true if multiple workers activate, false otherwise]" -Darg2="[true/false: true if prefetching activated, false otherwise]"

Clients:
2.3 Euclidean TSP: To run the Euclidean TSP task: ant runTspClient -Darg0="[ip of server machine]"

If no argument is passed with ant command to Clients EuclideanTSP or Computer, the program will use default value for server domain which is "localhost"
