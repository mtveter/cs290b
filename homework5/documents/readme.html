Readme - Project

Instructions to compile and run divide and conquer tsp with console GUI

Ant is used to both run and compile the application.

1. Compiling:
Run the command ant to compile program
The application has five packages:
- api
- clients
- gui
- models
- system
- tasks
- util

2. Running the code:

Space/server:
2.1 Start the space console GUI by running the command ant runConsole, and then press the button labeled "Start Space" before continuing to point 2.2

Computer(s):
2.2 Start computer server(s) by running the command ant runComputer -Darg0="[ip of server machine]" -Darg1="[true]" (multiple workers activate) -Darg2="[true]" (prefetching activated)

Clients:
2.3 Euclidean TSP: To run the Euclidean TSP task: ant runTspClient -Darg0="[ip of server machine]"

If no argument is passed with ant command to Clients EuclideanTSP or Computer, the program will use default value for server domain which is "localhost"
