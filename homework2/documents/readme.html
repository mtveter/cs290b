Instructions to compile and run your client and compute server

Ant is used to both run and compile the application.

1. Compiling:
Run the command "ant" to compile
The application has four packages:
- api
- clients
- system
- tasks

2. Running the code:

Server:
2.1 Start computer server by running the command ant runComputer

Client:
2.2 Mandelbrot: Start a client by running the command: ant runMandelbrotClient -Darg0="[ip of server machine]"
2.3 EuclideanTsp: To run the EuclideanTSP task, run: ant runEuclideanTsp -Darg0="[ip of server machine]"
