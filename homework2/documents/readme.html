Instructions to compile and run your client and compute server

Ant is used to both run and compile the application.

1. Compiling:
Run the command "ant" to compile program
The application has five packages:
- api
- clients
- system
- tasks
- util

2. Running the code:

Server(Space Implementation):
2.1 Start computer server by running the command ant runSpace
Computer:
2.2 Start computer server by running the command ant runComputer -Darg0="[ip of server machine]"

Clients:
2.3 Mandelbrot Set: Start a client by running the command: ant runMandelbrotSetClient -Darg0="[ip of server machine]"
2.4 EuclideanTsp: To run the EuclideanTSP task, run: ant runEuclideanTsp -Darg0="[ip of server machine]"

If no argument is passed with ant command to Clients for MandelbrotSet, EuclideanTSP or Computer, the program will use default value for server domain which is "localhost"
